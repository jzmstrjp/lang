## やってほしいこと

英語クイズの誤回答選択肢（incorrectOptions）を3つ生成してください。

## 背景

英語リスニングクイズの問題で、正解の日本語訳（japaneseSentence）に対する誤回答選択肢を作成します。

## 生成ルール

### 文字数の制約（最重要）

- **3文とも、確実にjapaneseSentenceとほぼ同じ文字数にしてください**
- 無理な場合は、japaneseSentenceより**少しだけ長く**してください（短くするのは禁止）
- 文字数が足りない場合は、自然な形で言葉を補ってください

### 内容の制約

1. **3文とも、senderRoleの立場でのセリフにすること**
2. **japaneseSentenceとは明確に意味が違う文を生成すること**
   - 似たような意味の文は禁止

3. **3文の多様性**
   - 1つ目: 馬鹿馬鹿しい文章
     - 例: 「当店で一番不人気な、とても苦いパフェはいかがですか？」
     - 例: 「全て期限切れの食材で作ったパフェはいかがですか！本日限定ですよ！」
   - 2つ目: japaneseSentenceと真逆の意味の文章
   - 3つ目: その他の誤答（状況に合わない文章など）

4. **文の始まり方**
   - 3文とも、**必ず違う語で始めること**（同じ語で始まるのは禁止）
   - japaneseSentenceと同じ語で始まることも禁止
   - 「まずは」「ちなみに」「ところで」などを文頭に加えて誤魔化すのは禁止

5. **疑問文の場合**
   - japaneseSentenceが疑問文の場合、incorrectOptionsの3文も**全て疑問文**にすること

### その他の注意点

- 全てのセリフはAI音声で読み上げるので、カッコ書きなどは含めず、AIが読み上げ可能な文字列にすること
- 自然な日本語であること

## 出力形式

以下のJSON配列形式で出力してください：

```json
["誤回答選択肢1", "誤回答選択肢2", "誤回答選択肢3"]
```
